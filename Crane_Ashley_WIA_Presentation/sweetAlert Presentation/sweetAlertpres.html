sweetAlert

What does it do?

sweetAlert is a javascript library that offers a great replacement for default alerts

Stand-alone or a plugin?

sweetAlert can be used with jQuery or as a stand-alone library!

Where can I get it?

http://tristanedwards.me/sweetalert

Why it should be considered

  • Default alerts look out of date
  • It's highly customizable.
  • It's simple to use.

Two different ways to call sweetAlert

  1. sweetAlert();
  2. swal();

Example using sweetAlert with jQuery


$("").click(function(){
    
        sweetAlert("Text String goes here!");
    
    
    });

Example using sweetAlert shortcut with jQuery


$("").click(function(){
    
swal("Text String goes here!");
    
    
    });
